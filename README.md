# AI-Based Automatic Exam Paper Generator

## Overview
This project is an **AI-powered system** that automatically generates exam papers, significantly reducing manual effort and improving efficiency in exam preparation. It leverages **Python**, **Flask**, **OpenAI API**, and **OCR** technology to process scanned documents and create structured exam papers.

---

## Features
- **Automatic Exam Paper Generation** using AI
- **OCR Integration** to extract questions from scanned documents
- **Role-Based Access Control (RBAC)** for administrators and faculty
- **Efficient and Time-Saving** workflow for exam preparation

---

## Technologies Used
- **Python** – Backend development
- **Flask** – Web framework for building the application
- **OpenAI API** – AI-based question generation
- **OCR** – Extract questions from scanned documents
- **HTML/CSS/JS** – Frontend templates and UI
